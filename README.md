# 📊 Swiggy Sales Performance Analysis & Interactive Excel Dashboard

## 🔍 Project Overview
This project focuses on analyzing Swiggy sales data using Microsoft Excel.  
The objective was to clean raw transactional data, perform detailed analysis using Pivot Tables, generate business KPIs, and build an interactive dashboard to visualize sales performance.

The entire project was completed using advanced Excel features without external tools.

---

## 🧠 Business Objective
The goal of this project was to:
- Analyze overall sales performance
- Identify top-performing cities and states
- Understand food category contribution
- Track monthly, weekly, and daily sales trends
- Build an interactive dashboard for business decision-making

---

## 🛠 Tools & Features Used
- Microsoft Excel
- Data Cleaning (handling blanks, formatting, data types)
- Pivot Tables
- Pivot Charts
- KPI Calculations
- Slicers & Filters
- Map Visualization
- Conditional Formatting
- Dashboard Design & Layouting

---

## 📂 Project Workflow

### 1️⃣ Data Collection
- Dataset sourced from a YouTube tutorial providing Swiggy sales data.

### 2️⃣ Data Cleaning
- Removed inconsistencies
- Standardized data formats
- Verified numerical columns
- Ensured data accuracy before analysis

### 3️⃣ Data Analysis
Using Pivot Tables, the following analyses were performed:
- Monthly Sales Trend
- Daily Sales Trend
- Weekly Sales Trend
- Sales by State
- Sales by Food Type (Veg / Non-Veg)
- Top 5 Cities by Sales
- Quarterly Sales & Orders

### 4️⃣ KPI Creation
The following key metrics were calculated:
- Total Sales (₹53.01M)
- Average Ratings (4.34)
- Ratings Count (5.59M)
- Total Orders (197.43K)
- Average Order Value (₹268.51)

### 5️⃣ Dashboard Development
An interactive dashboard was built featuring:
- KPI cards
- Line charts for trends
- Donut chart for category split
- Map visualization for state-wise sales
- Bar charts for city comparison
- Slicers for Month, Category, and State filtering

## 📊 Key Insights
- Sales show consistent monthly growth with peak performance during specific months.
- Veg category contributes 63% of total sales.
- Bengaluru is the highest revenue-generating city.
- Certain states significantly outperform others in order volume.
- Sales remain relatively stable across weeks with minor fluctuations.
